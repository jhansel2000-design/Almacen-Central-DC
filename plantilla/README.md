# Plantillas Excel — WMS Control

Coloca aquí los archivos de ejemplo para la demostración:

| Archivo | Módulo | Uso |
|---------|--------|-----|
| `Linea_de_Trabajo.xlsx` | Línea de trabajo | Plantilla con columnas WMS |
| `Ejemplo_export_lineas_trabajo.xlsx` | Línea de trabajo | Export real del sistema |
| `Ejemplo_diario_facturas_cliente.xlsx` | Facturas | Diario de facturas del cliente |
| Excel de **Operaciones** | Operaciones | Control de almacén / órdenes |
| Excel de **Productividad** | Productividad | Tabla dinámica por empleado |

Importación: **Administración → Datos Excel** (zona según tipo de archivo).
